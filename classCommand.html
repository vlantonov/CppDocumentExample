Detailed Description

Interface of method for command execution.

Decouples the object that invokes the operation from the one that knows how to perform it. Holds:

  • An object
  • A method to be applied to the object
  • The arguments to be passed when the method is applied.

Commands can delegate to any methods of a receiver. Command normally specifies a sender-receiver connection with a subclass.
